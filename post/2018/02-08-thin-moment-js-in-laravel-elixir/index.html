    Laravel Elixir使用webpack打包只引入moment.js的中文语言包

    <span title='2018-02-08 09:21:58 +0000 UTC'>二月 8, 2018</span>

    方法是使用webpack内置的ContextReplacementPlugin插件
    修改_gulpfile.js_
    先引入webpack

    const webpack = require('webpack')

    增加自定义webpack配置

    elixir(mix => {

    Elixir.webpack.mergeConfig({
        plugins: [
            new webpack.ContextReplacementPlugin(/moment[\\/]locale$/, /^\.\/(zh-cn)$/)
        ]
    });

    之后运行

    gulp

    会发现打包之后的js会明显变小
